# 0.3.2 — 2026-06-20

Framework-maintenance release — `@cyanheads/mcp-ts-core` `^0.10.6 → ^0.10.9` plus the script, skill, and config re-syncs that range carries. No server source or tool behavior changes.

## Changed

- **`@cyanheads/mcp-ts-core`** `^0.10.6 → ^0.10.9`.
- **`scripts/check-dependency-specifiers.ts`** (new) — devcheck step (`Dependency Specifiers`, `--no-dep-specifiers`) that hard-fails on floating specifiers (`latest`, `*`, pre-release dist-tags) in `package.json`'s dependency sections and `bun.lock`'s `workspaces` maps; static and local, runs in the default and `--fast` passes. ([cyanheads/mcp-ts-core#246](https://github.com/cyanheads/mcp-ts-core/issues/246))
- **`scripts/lint-packaging.ts`** — added check 10 validating plugin marketplace manifests (`.claude-plugin/plugin.json`, `.codex-plugin/plugin.json`, `.codex-plugin/mcp.json`): non-empty descriptions, unscoped machine name in display fields, and the full `package.json` name in the `npx -y` install arg. Gated by `devcheck.config.json` `packaging.pluginManifests`. ([cyanheads/mcp-ts-core#240](https://github.com/cyanheads/mcp-ts-core/issues/240))
- **`devcheck.config.json`** — added `packaging.pluginManifests: true` to enable the plugin-manifest checks above.
- **`scripts/build-changelog.ts`, `scripts/devcheck.ts`, `scripts/check-framework-antipatterns.ts`, `scripts/check-skill-versions.ts`** — fresh-scaffold guards: changelog `--check` and the git-dependent devcheck steps now skip cleanly outside a git repo / before the first version file exists; `check-skill-versions` skips a `SKILL.md` deleted from the worktree. ([cyanheads/mcp-ts-core#242](https://github.com/cyanheads/mcp-ts-core/issues/242), [#243](https://github.com/cyanheads/mcp-ts-core/issues/243), [#237](https://github.com/cyanheads/mcp-ts-core/issues/237))
- **Skills** re-synced from framework `0.10.9`: `add-tool` 2.14→2.15, `api-canvas` 1.6→1.7, `api-config` 1.7→1.8, `api-context` 1.8→1.9, `api-telemetry` 1.1→1.2, `git-wrapup` 1.2→1.4, `orchestrations` 1.3→1.4, `polish-docs-meta` 2.7→2.9.
