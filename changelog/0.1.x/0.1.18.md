# 0.1.18 — 2026-06-02

## Changed

- **`@cyanheads/mcp-ts-core`** `^0.9.16 → ^0.9.21` — picks up three framework fixes: per-request log context (HTTP transport now carries fresh request + trace/span IDs instead of frozen boot context); `fetchWithTimeout` strips query-string secrets (e.g. `?api_key=`) from error messages and logs; `withRetry` fails fast on non-retryable errors and `ctx.fail` auto-populates the `retryable` flag.
- **README** client-config server key renamed to full package name (`openfda-mcp-server` → `@cyanheads/openfda-mcp-server`) in all install snippets.
- **Skills** synced from framework `0.9.21`: `add-tool`, `add-service`, `api-canvas`, `api-context`, `api-linter`, `api-utils`, `design-mcp-server`, `release-and-publish` updated; `api-mirror` and `orchestrations` added.
- **`scripts/devcheck.ts`** updated; `scripts/check-skill-versions.ts` and `scripts/release-github.ts` added.
- **`release:github`** script added to `package.json`.
