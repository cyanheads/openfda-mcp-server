# 0.1.20 — 2026-06-05

## Added

- **`openfda_drug_profile`** — composite tool that resolves one drug name to its canonical FDA identity (generic name, NDC, RxCUI, SPL set ID) in a single query, then fans out in parallel across bounded per-drug openFDA endpoints and merges the results into a single profile ([#10](https://github.com/cyanheads/openfda-mcp-server/issues/10)). Sections: label highlights, adverse-event summary (top 5 reactions + serious-report count), recall history, Drugs@FDA approval record, and current shortage status. Best-effort per section — a miss returns `null` rather than failing the whole call. Resolves identity once to avoid the cross-endpoint identifier drift that breaks naive tool-chaining.
