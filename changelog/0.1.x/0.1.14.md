- **`manifest.json`** — MCPB bundle descriptor for one-click Claude Desktop install. Declares `OPENFDA_API_KEY` as an optional sensitive `user_config` field.
- **`.mcpbignore`** — excludes development artifacts from the MCPB bundle.

- **Install badges** (`npm install` + `bunx` one-liners) added to README.
