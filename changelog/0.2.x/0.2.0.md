# 0.2.0 — 2026-06-08

## Changed

- **`openfda_count` renamed to `openfda_count_values`** — tool name, file (`count.tool.ts` → `count-values.tool.ts`), and export symbol (`countTool` → `countValuesTool`) all updated. Behavior is unchanged; clients referencing `openfda_count` must update to the new name. No deprecated alias. ([#13](https://github.com/cyanheads/openfda-mcp-server/issues/13))

## Added

- **`openfda_drug_profile` surfaces `meta.fanOutKey`** — the canonical identifier the structured sub-queries (recall/enforcement, Drugs@FDA, shortage) run against is now returned in `meta.fanOutKey` and rendered in `format()`. Makes the identity reconciliation visible to callers without a second call. ([#12](https://github.com/cyanheads/openfda-mcp-server/issues/12))
- **`@types/node` `^25.9.1` → `^25.9.2`** — patch dev-dep update.
